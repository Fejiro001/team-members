# TeamOne - Team Members Website

## Description

TeamOne is a simple and responsive website that displays the current team members along with their profile pictures and roles. The project focuses on clean layout design, basic navigation, and structured content using HTML, CSS, and JavaScript.

## Pages & Sections

- **Header**
  - Website title: _TeamOne_
  - Navigation links: Home, About, Contact

- **Main Section**
  - Team Members list
  - Each member includes:
    - Profile image
    - Name
    - Role (MITT Software Developer Student)

- **Footer**
  - Basic container for future content

## Features

- Responsive layout using modern CSS
- Organized structure with reusable classes (`container`, `img-container`)
- Optimized images with alt text for accessibility
- External CSS and JavaScript files linked properly

## Technologies Used

- HTML5 (semantic structure)
- CSS3 (styling and layout)
- JavaScript (linked for future functionality)
